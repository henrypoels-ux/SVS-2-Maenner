# SVS-2-Maenner — Mannschaftskasse

Web-App zur Verwaltung der Mannschaftskasse der 2. Herren des SV Straelen.

## Funktionen
- Spielerverwaltung mit individuellem Saldo
- Strafenkatalog, Strafen & Gutschriften
- Theke-Saldo, Bargeld-Bewegungen und Ausgaben
- PayPal-→-Bank-Übertragung und sonstige Einnahmen
- Sammelstrafen / Sonderbeiträge
- PDF-Export (Spieler- und Monatsabrechnung)
- Passwortschutz

## Technik
Statische Single-Page-App (`index.html`, HTML/CSS/JS in einer Datei) mit **Supabase** als Backend. Deployment über **GitHub Pages**.

## Entwicklung
Änderungen an `index.html` vornehmen — GitHub Pages veröffentlicht den `main`-Branch automatisch.
